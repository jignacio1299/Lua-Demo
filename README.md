# Lua Demonstration
Class project for UIC's CS 341, Programming Language Concepts, taught by Shannon Reckinger

This repository is a collection of demonstrations for various Lua features:


calculator.lua -- demonstration of tables

countdown.lua  -- demonstration of coroutines

dog.lua        -- demonstration of OOP

factorial.lua  -- demonstration of metatables and __index

time.lua       -- demonstration of metatables and __add
